# Entregues Seminaris i Pràctiques SCAV

En aquest repositori, hi trobareu les quatre entregues corresponents a la part de vídeo de l'assignatura Sistemes de Codificació d'Àudio i Vídeo, de la Universitat Pompeu Fabra (UPF), curs 2025-26.

* Primera entrega: [Seminari 1](https://github.com/jordirubio01/SCAV/tree/main/01-S1). Script inicial, amb mètodes i funcions relacionades amb JPEG, JPEG2000 i FFMpeg.
* Segona entrega: [Pràctica 1](https://github.com/jordirubio01/SCAV/tree/main/02-practice1). Implementació de les funcionalitats anteriors amb FastAPI, utilitzant dockers amb serveis API i FFMPEG.
* Tercera entrega: [Seminari 2](https://github.com/jordirubio01/SCAV/tree/main/03-S2). Ampliació amb més endpoints per a vídeos.
* Quarta entrega: [Pràctica 2](https://github.com/jordirubio01/SCAV/tree/main/04-practice2). Ampliació d'algunes funcionalitats i implementació en una GUI.
